📐 Решение уравнения с параметром

\[ |x-a| + |x+a| = 2a + 3 \]

Найти все значения параметра \(a\), при которых уравнение имеет хотя бы одно решение, и выразить решения \(x\) в зависимости от \(a\).

📌 Шаг 1. Используем свойство суммы модулей

Для любых действительных чисел известно тождество:

\[ |x-a| + |x+a| = 2 \max\{|x|,\,|a|\} \]

(сумма расстояний до точек \(-a\) и \(a\) равна удвоенному большему из расстояний до нуля).

Подставляем в исходное уравнение:

\[ 2 \max\{|x|,\,|a|\} = 2a + 3 \quad\Rightarrow\quad \max\{|x|,\,|a|\} = a + \frac{3}{2}. \]

Правая часть должна быть неотрицательной, так как максимум модулей неотрицателен:

\[ a + \frac{3}{2} \ge 0 \;\Longrightarrow\; a \ge -\frac{3}{2}. \]

📌 Шаг 2. Разбор случаев: кто больше — \(|x|\) или \(|a|\)?

Случай А: \(|a| \ge |x|\). Тогда \(\max\{|x|,|a|\}=|a|\), уравнение становится:

\[ |a| = a + \frac{3}{2}. \]

Решаем с учётом знака \(a\):

  • Если \(a \ge 0\), то \(|a|=a\) и получаем \(a = a + 1.5 \;\Rightarrow\; 0=1.5\) — противоречие, решений нет.
  • Если \(a < 0\), то \(|a| = -a\) и \(-a = a + 1.5 \;\Rightarrow\; -2a = 1.5 \;\Rightarrow\; a = -0.75\).

Значение \(a = -0.75\) удовлетворяет условию \(a<0\) и \(a \ge -1.5\). При этом условие \(|a| \ge |x|\) даёт \(|x| \le |a| = 0.75\). Подстановкой в исходное уравнение убеждаемся: для любого \(x\) с \(|x| \le 0.75\) левая часть равна \(2\cdot0.75 = 1.5\), правая \(2(-0.75)+3 = 1.5\).

\[ \text{При } a = -0.75 \text{ решения: } x \in [-0.75;\,0.75] \quad (\text{все } |x| \le 0.75). \]

📌 Шаг 3. Случай Б: \(|x| \ge |a|\)

Тогда \(\max\{|x|,|a|\}=|x|\) и уравнение принимает вид:

\[ |x| = a + \frac{3}{2}. \]

Поскольку левая часть неотрицательна, условие \(a \ge -1.5\) уже выполнено. Отсюда:

\[ x = \pm\left(a + \frac{3}{2}\right). \]

Дополнительное требование \(|x| \ge |a|\) даёт неравенство:

\[ a + \frac{3}{2} \ge |a|. \]

Рассмотрим знаки \(a\):

  • При \(a \ge 0\): \(a + 1.5 \ge a\) верно всегда.
  • При \(a < 0\): \(a + 1.5 \ge -a \;\Rightarrow\; 2a + 1.5 \ge 0 \;\Rightarrow\; a \ge -0.75\).

Объединяя с условием \(a \ge -1.5\), получаем для случая Б: \(a > -0.75\) (значение \(a = -0.75\) даёт \(|x|=0\), но тогда \(|x| \ge |a|\) не выполняется, так как \(0 \ge 0.75\) ложно). Итак, при \(a > -0.75\) имеем два решения:

\[ x = a + \frac{3}{2}, \qquad x = -\left(a + \frac{3}{2}\right). \]

При этом \(a+1.5 > 0\), поэтому оба корня различны.

📌 Шаг 4. Объединение результатов

Проанализируем все \(a\) из области \(a \ge -1.5\):

  • При \(a < -0.75\) (включая \(a=-1.5\)) нет ни одного решения: случай А даёт только \(a=-0.75\), случай Б требует \(a \ge -0.75\).
  • При \(a = -0.75\): бесконечное множество решений (отрезок).
  • При \(a > -0.75\): два симметричных решения \(x = \pm (a+1.5)\).

Кроме того, при \(a < -1.5\) правая часть \(2a+3\) отрицательна, а левая часть (сумма модулей) неотрицательна → решений нет.

✍️ Ответ

1. Значения параметра \(a\), при которых уравнение имеет хотя бы одно решение:

\[ a \in [-0.75,\; +\infty) \]

2. Выражение решений \(x\) в зависимости от \(a\):

  • Если \(a = -0.75\), то любое \(x\), удовлетворяющее \(|x| \le 0.75\) (то есть \(x \in [-0.75;\,0.75]\)).
  • Если \(a > -0.75\), то ровно два решения: \[ x_1 = a + \frac{3}{2}, \qquad x_2 = -\left(a + \frac{3}{2}\right). \]

При \(a < -0.75\) (и \(a \ge -1.5\) тоже) — решений нет. При \(a < -1.5\) также нет решений.

🔍 Проверка на примере

Возьмём \(a = 0\): уравнение \(|x|+|x| = 2|x| = 3\) → \(|x| = 1.5\) → \(x = \pm 1.5\). Формула даёт \(x = \pm(0+1.5) = \pm 1.5\). ✅

Возьмём \(a = 2\): по формуле \(x = \pm 3.5\). Подстановка: \(|3.5-2|+|3.5+2| = 1.5+5.5=7\); правая часть \(2\cdot2+3=7\). ✅

Возьмём \(a = -0.75\): отрезок \([-0.75;0.75]\), например \(x=0.5\): \(|0.5+0.75|+|0.5-0.75| = 1.25+0.25=1.5\); правая часть \(2(-0.75)+3=1.5\). ✅
